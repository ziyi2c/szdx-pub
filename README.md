# szdx 公共模板

## 说明

> 1. 这是一个开源课设模板，仅限用于科成的好兄弟学习使用。
> 2. 由于这是个模板，且由于服务器资源以及数据保密性等原因，故未提供用户名及密码。
> 3. 用户名及密码未提供注册渠道，请**联系开发者获取**。
> 4. js 逻辑仅供参考，请自行设计样式！！！
> 5. 老师知道模板的事情，所以实在需要用模板的，一定自己改改样式，因为这是很简单而且很有意思的事情。
> 6. 欢迎 Star 支持开源。

# 使用 


## 安装依赖包
```
npm install
```

### 热部署
```shell
npm run serve
```

## 一步到位

```shell
npm i && npm run serve
```



# 注意

> 装包失败请检查你电脑的权限！！！
>
> 好多人的电脑是开启UAC了的，VScode的终端**有可能**没权限，此时**尝试管理员权限打开命令行**再执行命令！！！
